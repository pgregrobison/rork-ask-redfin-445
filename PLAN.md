# Refine Home Card Layout & Icons

## Changes

**Single Badge Over Photo**
- Home cards and map popup cards will only ever show one badge at a time
- Priority order: "Xd ago" → "HOT HOME" → "LISTED BY REDFIN"
- This applies everywhere a home card appears (map popup, list, For You, Ask Redfin)

**Tags Row — Single Line Only**
- The facts/tags below the address will be constrained to a single line
- If one tag is too long and would cause wrapping, fewer tags will be shown to keep everything on one row

**Fix Map Card Info Centering**
- The price, stats, address, and tags on the map popup card will stretch to fill the full card width (left-aligned) instead of appearing pushed toward the center

**Reduce Excessive Padding**
- Map popup cards: tighten internal info padding so content isn't pushed inward
- List view cards: reduce right-side padding so content fills the card width naturally

**Medium Icons on Home Cards**
- The heart and share icons on home cards will be updated from small (15pt) to medium (17pt) size with matching tap targets

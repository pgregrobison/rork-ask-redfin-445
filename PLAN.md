# Swap Find tab icon to binoculars and restyle Edit profile button

## Changes

- **Find tab icon**: Replace the magnifying glass with the binoculars symbol for both the default and selected states of the Find tab.
- **Edit profile button**: Restyle the "Edit profile" button on My Redfin to a tertiary outline style — a transparent pill with a thin border and primary-colored label — instead of the current solid filled pill. It will reuse the existing outlined treatment used elsewhere in the app so it adapts cleanly to light and dark mode.

No other tabs, screens, or buttons are affected.

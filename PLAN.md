# Match launch screen background to Redfin red

**What changes**

- Update the brief moment after tapping the app icon (before the splash animation appears) so it shows the same Redfin red as the splash background instead of black.
- The splash logo and animation remain exactly the same — only that initial flash of color changes.

**Result**

- Tapping the app icon now feels seamless: red flash → red splash with logo → app, with no jarring black frame in between.

# Detail Page Styling Updates — James Style

## Changes

### Photo Area
- Add **12pt spacing** above the segmented control for the carousel dots
- Apply **liquid glass effect** to the segmented control so it's visually distinct against the photo background
- Add **icons alongside text** in the segmented control: play.fill (Media), mappin.fill (Map), cube.fill (3D), binoculars.fill (Street)

### Price & Stats
- Update beds/baths/sq ft text to use a **larger, bolder style** (~18pt semibold) in system primary color (white in dark, black in light)

### Rate Summary (New — above Request Showing)
- Add a compact rate summary section between price/address and the "Request showing" button
- Shows estimated monthly payment (e.g. "$2,612/mo estimated") with an "Estimate my rate" full-width primary button

### Request Showing
- Stays in its current position, right below the new rate summary

### Property Details Section (First Container)
- Returns to the **left-icon, right-info row style** — icon on the left, value + label stacked on the right
- Covers: property type, year built, lot size, price per sq ft, Redfin estimate, HOA dues
- Uses **24pt internal spacing** between rows

### Feature Highlights (New Section — inside same container)
- A **2×3 grid** of home feature highlights with associated icons (e.g. "Stunning island", "2 walk-in closets")
- Always shows exactly **6 items** — fills from listing tags first, then auto-generates highlights from listing data (beds, year built, sq ft, etc.)
- Each cell: icon above, tag label below

### Description & Full Details (Moved into container)
- The text description and "Full property details" button move **inside the same container** as the feature highlights grid, below a divider

### Container Internal Spacing
- All containers updated to use **24pt gap** between elements for more breathing room

### "Let's Chat" Button
- Changed to a **full-width secondary (outlined) button**

### Lifestyle Section
- **Remove the top center decorative icon** (the large walking figure)
- Grid of icon+tag cells remains as-is

### General

# Default to Accessory + Map Focus

Change the app's default experience so new users (and anyone without a saved preference) start with:

- **Accessory** layout — the native tab bar with the Ask Redfin input pinned at the bottom
- **Map Focus** search behavior — chat drops to a half-sheet on search and pins auto-fit on the map

Existing users who've already picked something else in the debug panel will keep their saved choice.
